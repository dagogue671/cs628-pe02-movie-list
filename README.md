This program is a react app that displays a list of movies. A component named MovieList.js is exported to the App.js file for rendering. The MovieList.js file extracts data from an array object in the movies.js file. It takes the data maps the data using a useState condition. The useState by is set to 'All Genres' by default to display all movies, but will filter once a different options is selected in it's dropdown list. The movies are seperated by cards, which hold an onClick event. When the card container is clicked, an alert message will pop up.

To run program, cd into the movie-list folder and run the app using npm start
```
cd movie-list
npm start
```
